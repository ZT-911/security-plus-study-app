Data Protection — Theory

Data Classification

Defines how sensitive data is and how it must be protected. Levels include Public, Private, Confidential and Restricted. The more sensitive the data, the stronger the controls required.

Data States

Data exists in three states: At Rest (stored), In Transit (moving), and In Use (being processed in memory). Data in Use is the hardest to protect.

Data Loss Prevention (DLP)

DLP systems detect and prevent data exfiltration. They monitor sensitive information and block unauthorized transfers.

Encryption Basics

Symmetric encryption uses one key and is fast (AES). Asymmetric encryption uses public/private keys (RSA). Encryption protects confidentiality.

Data Policies

Define how data is stored, accessed, retained and destroyed. Includes retention policies and access control rules.

Data Destruction

Methods include clearing (logical), purging (secure erase) and physical destruction of media.
