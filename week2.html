Data Classification
Understand how data sensitivity determines protection and handling requirements.
Core Concept

Data classification defines how sensitive information is and determines the level of protection, access control, storage, and handling required.

Common Classification Levels
Public

No impact if disclosed

Private / Internal

Internal use only

Confidential

Sensitive data requiring protection

Restricted

Severe impact if exposed

Key Principle

The more sensitive the data, the stronger the required controls and handling rules.

Exam Insight

Security+ questions often require matching the correct control or access level to the appropriate data classification.
